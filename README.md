# 🌸 Pudra Beauty Room — Website

**Курсовий проект | React Single-Page Application**

Сучасний багатосторінковий сайт для студії апаратної косметології *Pudra Beauty Room* у Нововолинську.

---

## 🚀 Запуск проекту

### Встановлення залежностей

```bash
npm install
```

### Запуск у режимі розробки

```bash
npm start
```

Відкрийте [http://localhost:3000](http://localhost:3000) у браузері.

### Збірка для продакшн

```bash
npm run build
```

## ✨ Функціонал

### Інтерактивні елементи
- 🔵 **Кастомний курсор** — рожева крапка + обводка, реагує на hover
- 📜 **Scroll progress bar** — лінія прогресу прокрутки зверху
- ⏳ **Лоадер** — анімований прелоадер з прогрес-баром при першому відкритті
- 🌐 **Smooth transitions** — плавні переходи між сторінками (Framer Motion)

### Анімації
- Parallax-ефект на Hero секції
- Fade-in + slide-up при скролінгу (IntersectionObserver)
- 3D-сфера на головній (Canvas API, радіочастотний градієнт, частинки)
- Плаваючі картки-статистики на головній
- Marquee рядок з послугами
- Анімовані лічильники статистики
- Hover-ефекти на всіх картках та кнопках

### Сторінки
1. **Головна** — Hero з 3D-сферою, послуги, статистика, переваги, відгуки, CTA
2. **Послуги** — картки з фільтрацією по категоріях
3. **Деталі послуги** — повний опис, переваги, процедура, протипоказання, сайдбар
4. **Про нас** — місія, цінності, команда, сертифікати
5. **Прайс** — таби по категоріях, таблиця цін
6. **Галерея** — masonry grid, лайтбокс
7. **Контакти** — форма, соцмережі, карта-плейсхолдер
8. **Запис** — 3-кроковий wizard (послуга → дата/час → контакти)
9. **404** — стилізована сторінка помилки
10. **500** — сторінка серверної помилки
